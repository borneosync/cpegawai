## 📖 Tentang Aplikasi

CPegawai merupakan aplikasi manajemen kepegawaian yang dikembangkan untuk membantu instansi pemerintah maupun organisasi dalam mengelola data pegawai secara digital.

Aplikasi ini hadir sebagai pengembangan dari konsep aplikasi kepegawaian sebelumnya dengan berbagai penyempurnaan pada sisi fitur, kemudahan penggunaan, keamanan, dan efisiensi pengelolaan data.

---

## ✨ Fitur Utama

- 👤 Manajemen Data Pegawai
- 🆔 Data NIP dan Identitas Pegawai
- 🏢 Data Jabatan dan Unit Kerja
- 📋 Riwayat Pangkat dan Golongan
- 🎓 Riwayat Pendidikan
- 📄 Manajemen Dokumen Pegawai
- 🔍 Pencarian Data Cepat
- 📊 Dashboard Statistik Pegawai
- 🔐 Hak Akses Berbasis Role
- 📱 Responsive Design (Mobile Friendly)
- 🔔 Notifikasi Sistem
- 📥 Export Data

## 📦 Instalasi

### 1. Clone Repository

```bash
git clone https://github.com/borneosync/cpegawai.git
```

### 2. Masuk ke Folder Project

```bash
cd cpegawai
```

### 3. Salin File Environment

```bash
cp env .env
```
